2.0 - Project - Hello World!

Alright so very quickly we can start adding to this canvas using Phaser's functions to do most of the work for us!

Displaying Text:


game.add.text(positionX, positionY, text, style)

  • positionX: Horizontal position of the text
  • positionY: Vertical position of the text
  • text: Text to display
  • style: Object that states the style of the text
var mainState = {
     preload: function()
     {
           //This function will be executed at the beginning
           //This is where we load the game's assets 
     },
     create: function()
     {
           //This function is called after the preload function
           //Here we set up the game, add sprites, add spritesheets, animations etc.
           this.helloText = game.add.text(250, 170, 'Hello World!', {font: '20px Roboto', fill: '#FF0000' } );
     },
     update: function()
     {
           //This function is called 60 times per second
           //It contains the game's logic
     }
};

var game = new Phaser.Game(500, 340, Phaser.AUTO, 'game');
//Then we must use this mainState variable instead of an inline object
game.state.add('main', mainState);

game.state.start('main');

This one line of code inside the create function adds text to our canvas!
