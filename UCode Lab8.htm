4.5 - Lesson - Loops

Another common thing to do in JavaScript is having to go through an entire array to look at all of its data. This is solved using loops!

  • We will learn three different loops to loop over the same array to see all of its data:

For Loop:

//First let's make an array, it can contain any data you want!
var yourArray = ["UCode", "is", "awesome", 9001, 4, true, false, {firstName: "Super", lastName: "UCoder"}];

//Now we need to loop through the array to get it's data, one element at a time!

//The first loop I will use is called a For Loop, which has three steps to write it. 
//The first is the initialization of an iterator. Usually a variable that starts at 0.
//The second is the condition. This condition usually checks the iterator and is checked after each loop, if it is true the loop will continue, if it is false the loop will end!
//The third and final thing is the actual iteration of the variable iterator. You add or subtract to the iterator.

//Create a var i (for iteator) and initialize it to 0, i < length of the array (array.length returns the number of elements in the array, i++ adds one to the variable i
for(var i = 0; i < yourArray.length; i++)
{
   console.log(yourArray[i]); //notice I use the variable i for the index into the array. 
   //We like to start at 0 for iterators because arrays indices start at 0 and it creates a perfect index for us!
}
//Congratulations, you wrote a for loop accessing an array's data!

While Loop:

//Now, lets write a while loop! 
//A while loop has one condition that it checks, if the condition is true it will keep looping, if it is false the loop will stop.

var i = 0; //we still need an iterator so lets declare one at the top of our file
while(i < yourArray.length)
{
    console.log(yourArray[i]); //Will log the data at index i
    i++;  //We still need to iterate and add to our iterator so it does not log the same index over and over!
}
//As you can see, the three steps of the for loop are now separated in their own lines in the while loop, and both loops are doing the exact same thing.
//Congratulations you wrote a while loop accessing an array's data!

Do While Loop:

//Now for the final loop!
//A do while loop!
//This loop is basically a while loop that is separated into the code block that will be looped (put inside the `do` part), and the condition that is checked (put inside the `while` part).

var i = 0;

do {
   console.log(yourArray[i]);  //will log data at index i
   i++;  //will still need to add to the index to change which element to access
}
while(i < yourArray.length);

//Congratulations you know now the three different loops in JavaScript
