Welcome to Phaser.js UCoder!

HTML Code:

This code does two important things:

  • Loads all the JavaScript files: Phaser.min.js and main.js
  • Adds a div with an id of game that will contain the game itself!
<!doctype html>
<html>
<head>
<meta charset="utf-8">
<title> Mario UCoin Game! </title>
<script type="text/javascript" src="//js.ucode.com/phaser.min.js"></script>
<script type="text/javascript" src="main.js"></script>

</head>

<body>
          <div id="game">

          </div>
</body>
</html>
  • Right away save this as a Phaser Template HTML file, it will be useful to re-use this HTML file instead of re-writing the same exact file to start each time!

If you wish to get all the assets at once there is a simple way to do it!
If you wish to create your own assets or just get them at the time we need them, then skip this next step!


Begin by opening up the Terminal.

In Codio it would be Tools->Terminal (Shift-Alt-T), Cloud9 already has a terminal window shown.
Type the following inside the Terminal and push Enter:

wget troycurnel.com/phaser/assets.zip

Awesome, now you should have an assets.zip inside your master folder! We need to unzip it through the Terminal
Type the following inside the Terminal and push Enter: 

unzip assets.zip

There we go, we have all our assets for the game now!
