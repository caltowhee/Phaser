3.0 - Project - Add A Sprite

What function do we use to load a sprite?

game.load.image(imageName, imagePath);
game.load.img(imageName, imagePath);
game.load.image(imagePath, imageName);
game.load.sprite(imageName, imagePath);

Alright, if you are at this point then you have successfully drawn out text to the canvas. You guys are doing great!

Now you are ready to add an image, Phaser as well as other 2D frameworks call an image a sprite!
Lucky for you, we have provided you with an assets folder that contains many images, sprite-sheets, and sounds!

  • In Phaser, everytime we want to use an asset (image, sound, etc.) we first need to load it. For an image we can do that with game.load.image function:

Load the Sprite:

game.load.image(imageName, imagePath)

  • imageName: the name (key) that will reference the image/sprite.
  • imagePath: the path to the image.
    preload: function()
    {
         //Here we load the image:
         game.load.image('mario', 'assets/mario.png');       
    }


  • Once the sprite is loaded, we can display it on the screen with game.add.sprite function.

Display the Sprite:

game.add.sprite(positionX, positionY, imageName)

  • positionX: Horizontal position of the sprite.
  • positionY: Vertical position of the sprite.
  • imageName: The name of the image, predefined in the game.load.image first parameter.
    create: function()
    {
         this.helloText = game.add.text(250, 170, 'Hello World', { font: '20px Roboto', fill: '#FF0000' } );
         this.sprite = game.add.sprite(game.world.centerX, game.world.centerY, 'mario');
    }



Rotate the Sprite

  • Now that a sprite has been added to our game/canvas we can start manipulating it! Let's start by rotating it one degree every frame!

Sprite's have many member variables that access important data, one of these is it's angle! 

    update: function()
    {
         this.sprite.angle++;
    }
