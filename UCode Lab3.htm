1.25 - Lesson - Associated arrays

We will be learning about Associated Arrays and how useful key|value pairs are in JavaScript programming!

  • A key|value pair is used very frequently in programming and you need to know this! It is a simple concept, a key is a short name that refers to the value; almost like a nickname such as Alex short for Alexander.

  • However, in JavaScript programming the value tends to be a very large Object or Number with a key being a short String.

  • Let's demonstrate this functionality in a hypothetical situation: Let's say Mario and his friends just took a test and have a possible score of 0 to 100!

var grades = {

mario: 99,
luigi: 98,
koopa: 88,
bowser: 85,
peach: 100,
daisy: 90   

};


//Now since we have a structure of keys as names with values as grades we can just call the key to get the associated value!
//Here are two ways we can access the key to get the value!
console.log(grades["mario"]); //Treat it like an array and instead of an index we pass in the key!
console.log(grades.mario);  //Treat the key like a member variable to access the data!
//This will log Mario's associated grade which is 99!

console.log(grades["peach"]); //Treat it like an array and instead of an index we pass in the key!
console.log(grades.peach); //Treat the key like a member variable to access the data!
//This will log Peach's associated grade which is 100!

//You can do this with any member variable with any type you have added to the object!

Create your own associated array with 4 or more key + values and console.log each key to get the associated value!
